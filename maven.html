Maven

You can use Glimpse with maven by adding the following to your pom.xml:

<dependencies>
    <dependency>
        <groupId>com.metsci.glimpse</groupId>
        <artifactId>glimpse-core</artifactId>
        <version>2.1.1</version>
    </dependency>
</dependencies>

The full Glimpse release contains a number of modules which add additional features (and usually additional third-party dependencies) to Glimpse. Each has its own Maven artifact:

        <artifactId>glimpse-extras-charts</artifactId>
        <artifactId>glimpse-extras-dspl</artifactId>
        <artifactId>glimpse-extras-spacenav</artifactId>
        <artifactId>glimpse-extras-swt</artifactId>
        <artifactId>glimpse-extras-examples</artifactId>

All the non-extras modules are available directly from Maven Central. In most cases, this is all you will need. However, some of the extras modules contain dependencies which are still only available through the Metron Maven Repository. Also, older releases are currently only available through the Metron Maven Repository.

In order to download extras artifacts or older versions, you must add Metron's Maven repository to your project's pom.xml or your settings.xml:

<repositories>
    <repository>
        <id>glimpse-releases</id>
        <url>http://glimpse.metsci.com/maven2/metsci-releases</url>
    </repository>
    <repository>
        <id>glimpse-snapshots</id>
        <url>http://glimpse.metsci.com/maven2/metsci-snapshots</url>
    </repository>
    <repository>
        <id>glimpse-thirdparty</id>
        <url>http://glimpse.metsci.com/maven2/thirdparty</url>
    </repository>
</repositories>

For more information, see the Maven documentation on Setting up Multiple Repositories.
