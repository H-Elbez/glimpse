Download

Glimpse is released in two flavors: core and full. In most cases, core contains everything you need.

The full release contains Glimpse extensions for integrating Glimpse with SWT, 3D Mice, Google DSPL, and NOAA ENC charts. If you are using Maven you can depend on the various Glimpse extensions individually.

Binary

Download the latest release: glimpse_core_1.2.0.zip or glimpse_full_1.2.0.zip.

Download older releases: https://github.com/metsci/glimpse/downloads.

Depend on Glimpse using Maven.

Source

Download the latest source in either zip or tar formats.

Get the source code by forking us on GitHub: metsci/glimpse.
