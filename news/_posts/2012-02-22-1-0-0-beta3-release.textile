We're excited to announce the open source launch of the Glimpse data visualization library! You can fork the code on <a href="https://github.com/metsci/glimpse">GitHub</a> or check out the <a href="/download.hatml">Download</a> page for binary releases.

Glimpse is a Java library which enables developers to easily construct interactive applications which take advantage of OpenGL and GPU hardware like programmable shader pipelines and cached texture memory to display large quantities of data at interactive speeds.
